# 📱 InvenStory: Android Inventory Management App  

## 📖 Project Overview  
This project was developed using **Android Studio** and focuses on building a native **Inventory Management Mobile Application** for Android devices.  

InvenStory allows users to create accounts, securely log in, and manage inventory items through full CRUD functionality (Create, Read, Update, Delete). Inventory items are displayed in a clean grid-style layout, and the application includes optional SMS notifications for low-stock alerts.  

The project emphasizes **mobile application architecture**, **database persistence**, **runtime permission handling**, and modern **Android development practices**.

## 📂 Project Structure  
- `app/src/main/java/` – Java source files (Activities, database helpers, logic).  
- `app/src/main/res/layout/` – XML layout files for UI components.  
- `app/src/main/AndroidManifest.xml` – Application configuration and permission declarations.  
- `app/src/main/res/drawable/` – App icon and graphical assets.  

---

## 🚀 How to Run  
1. Clone the repository.  
2. Open the project in **Android Studio**.  
3. Build and run using an Android Emulator (API 29+) or physical Android device.  
4. Create an account and begin managing inventory items.
